# @renovacell/landing — Landing pública + Agente IA (Módulo 1)

Carpeta reservada para la **landing pública** de Renovacell (sitio de marketing,
catálogo, captación de prospectos con agente IA).

**Aún vacía a propósito.** El código vive hoy en un repo aparte
(`github.com/Davidespinozan/renovacell`) y se traerá aquí con `git subtree`
—preservando su historial— cuando arranquemos el **Módulo 1 (Fase 1: Comercial)**.

Compartirá `@renovacell/ui` (diseño) y `@renovacell/db` (datos) con `apps/web`,
pero tendrá su propio deploy en Netlify (base directory `apps/landing`).
